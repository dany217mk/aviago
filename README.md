# Aviago  
Система управления чартерными авиаперелётами и онлайн-регистрацией пассажиров

Aviago — это веб-платформа, позволяющая автоматизировать организацию чартерных рейсов, управление ресурсами авиакомпаний и процесс покупки/регистрации пассажиров.  
Проект разработан в рамках университетских дисциплин («ООП» и «Проектирование и реализация БД»).

## Описание системы

Платформа объединяет нескольких участников процесса чартерных авиаперелетов:

### **Заявитель**
- создаёт заявку на перелёт  
- выбирает маршрут, даты, авиакомпанию (или оставляет выбор системе)  
- может закрыть продажу билетов для сторонних пассажиров  
- получает уникальный код для группового бронирования  

### **Планировщик авиакомпании**
- получает новые заявки  
- создаёт рейс на их основе  
- назначает самолёт и экипаж  
- формирует расписание  

### **Пассажир**
- покупает билет на открытый рейс  
- может использовать уникальный код для группового бронирования  
- проходит онлайн-регистрацию  
- выбирает место  

### **Экипаж**
- получает через систему список назначенных рейсов  
- видит маршрут, тип ВС, список пассажиров  

### **Директор авиакомпании**
- управляет аккаунтами сотрудников  
- контролирует выполнение заявок и рейсов

## Основная функциональность

### Управление заявками
- создание заявок с маршрутом, датами и количеством мест  
- выбор авиакомпании или автоматическое распределение  
- настройка приватности продаж  

###  Планирование рейсов
- создание рейса на основе заявки  
- назначение самолёта и экипажа  
- автоматическое формирование расписания  

###  Билеты и бронирование
- индивидуальная покупка билетов  
- групповые бронирования по уникальному коду  
- разделение квоты мест между заказчиком и открытой продажей  

###  Онлайн-регистрация
- выбор мест  
- формирование списков пассажиров на посадку  

### Управление авиакомпаниями
- сотрудники и роли  
- список самолётов  
- рейсы  
- экипажи

## Как запустить локально

```sh
git clone https://github.com/dany217mk/aviago
```
1. Установить XAMPP или OpenServer
2. Установить PostgreSQL
3. Создать базу данных
4. Импортировать файл aviago_backup.sql
5. Создать файл config/constants.php (по примеру constants.example.php)
6. Включить Apache + PostgreSQL сервер
